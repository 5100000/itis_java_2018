# Lesson 02

## Строки

* String - строковый тип данных. `String s = "hello";`
* Чтобы считать строку - метод `next()` из класса `Scanner`
* Строки сравниваются специальным образом - через equals. Пример:
```
Scanner sc = new Scanner(System.in);
String s1 = "abc";
String s2 = sc.next();
if (s1.equals()) {
    //этот код выполнится именно тогда, когда в s2 тоже "abc"
}
```

## boolean

* boolean - логический тип данных. Переменные типа boolean
принимают значение true или false.
```
boolean a = true;
System.out.println(a);//на экран будет выведено true
```

## Массив

* Массив - структура данных, которая хранит набор значений
одного типа, где каждый элемент имеет свой индекс.\
При объявлениее массива необходимо указать число элементов в
массиве. После объявления массива нельзя изменить количество
элементов в нем. Пример:
```
int[] arr = new int[5];//объявление массива с именем arr из 5-и элементов
arr[2] = 3;//в ячейку с индексом 2 записали значение 3
```

## Цикл for

* for - цикл, содержит три элемента при объявлении: действие,
выполняемое один раз при заходе в цикл; условие, проверяемое каждый
раз перед следующей итерацией; действие, выполняемое после
каждой итерацией.\
Итерация - одно выполнение инструкций тела цикла (одно повторение)
